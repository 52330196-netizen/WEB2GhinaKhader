# Travel Destination - React Web Application

A modern, responsive React web application for exploring travel destinations worldwide.

## 📋 Project Description

Travel Destination is a frontend web application built with React that allows users to:
- Explore countries, cities, and attractions
- Search and filter attractions by category, rating, and price
- Add attractions to their wishlist
- Contact the travel agency
- Responsive design works on desktop and mobile devices
